Ayuda de MotoGP-Desktop

Inicio

Esta es la sección principal de la página web. Aquí encontrarás varias fotos dinámicas relacionadas con el circuito de Moto-GP Sachsenring. Si esperas unos segundos podrás ver como la foto cambia. Debajo de las imágenes verás las últimas noticias relacionadas con el campeonato del mundo de MotoGP.

Piloto

En esta sección encontrarás información sobre el piloto Fabio Di Giannantonio, así como algunos conceptos sobre MotoGP y enlaces de interés.

Circuito

En esta sección podrás cargar distintos archivos de texto para visualizar la información que contienen. Los tipos de archivos aceptados son HTML, SVG y KML; para abrirlos pulsa el botón "Seleccionar archivo" correspondiente y escoge el archivo que quieras mostrar. Si quieres visualizar otro archivo, simplemente selecciona el que te interese y se cargará a continuación.

Meteorología

En esta sección encontrarás información general sobre la ciudad de Oberlungwitz, en Alemania. Esta ciudad es la que se encuentra más próxima al circuito Sachsenring, del cual encontrarás más información en la sección "Circuito". Debajo de dicha información podrás ver datos meteorológicos de la ciudad el día en que tuvo lugar la carrera, así como los días de entrenamientos previos a ésta.

Clasificaciones

En esta sección podrás consultar información relativa a la clasificación de la carrera de MotoGP. Se muestra el ganador de la carrera, así como su tiempo empleado para completarla, y los tres primeros clasificados.

Juegos

En esta sección encontrarás varios juegos relacionados con Moto-GP, enumerados en una lista. Podrás elegir el juego que quieras pulsando su nombre dentro de dicha lista.

Juegos: Juego de memoria con cartas

En este juego se presentarán varias cartas con imágenes ocultas de logos relacionados con MotoGP. Cada imagen se está repetida, de manera que haya parejas de logos. Deberás ir pulsaldo las cartas para que revelen las imágenes y así encontrar todas las parejas de logos. En la parte superior hay un cronómetro que te indicará el tiempo que has empleado para encontrar dichas parejas.

Juegos: Cronómetro

En este juego podrás utilizar un crónometro que incluye minutos, segundos y décimas de segundo. Utiliza el botón de "Arrancar" para que el cronómetro empiece a correr, "Parar" para detener el cronómetro y "Reiniciar" para resetearlo.

Juegos: Cronómetro PHP

Este juego es similar al otro cronómetro, la diferencia es que no verás transcurrir el tiempo. Utiliza los botones de "Arrancar"y "Parar" para iniciar y detener el cronómetro respectivamente, y después podrás ver el tiempo que ha transcurrido pulsando el botón "Mostrar tiempo".

Juegos: Configuración Test

Este apartado está dirigido únicamente al observador encargado de llevar a cabo las pruebas de usabilidad. Hay 3 opciones: reiniciar la base de datos, borrando todo su contenido; eliminar por completo la base de datos o exportar los datos de la misma.

Juegos: Test

Este apartado contiene una serie de preguntas relacionadas con el proyecto. Puedes encontrar las respuestas en las distintas secciones de la página. Una vez que completes el cuestionario, deberás rellenar varios datos para el procesamiento de las respuestas.
